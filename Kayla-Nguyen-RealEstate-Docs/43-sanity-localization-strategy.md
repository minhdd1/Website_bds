# Sanity Localization Strategy

Approach:

Field-Level Localization

Reason:

- easier content management
- fewer documents
- easier references
- easier GROQ queries

Supported Field Types:

localeString

localeText

localePortableText

Example:

title:
{
  vi: "Một buổi sáng ở Thảo Điền",
  en: "A Morning in Thao Dien"
}
